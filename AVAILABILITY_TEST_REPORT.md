## 📋 下一步行动

### 立即可以做的
```bash
# 1. 安装前端依赖
cd frontend && npm install

# 2. 安装后端依赖
cd backend && pip install -e .

# 3. 启动数据库和 Redis
docker-compose up -d db redis

# 4. 运行数据库迁移
cd backend && alembic upgrade head

# 5. 启动后端服务
cd backend && uvicorn app.main:app --reload

# 6. 启动前端开发服务器
cd frontend && npm run dev
```
